# Python Inventory System

A console-based inventory management application built with Python.

## Features

* Add products
* List products
* Find products by ID
* Update product quantities
* Remove products
* Save data to JSON
* Load data from JSON

## Technologies

* Python 3
* JSON
* Git
* Object-Oriented Programming (OOP)

## Project Structure

```text
python-inventory-system/
├── data/
│   └── products.json
├── src/
│   ├── main.py
│   ├── product.py
│   └── inventory.py
├── tests/
├── README.md
└── requirements.txt
```

## How to Run

```bash
source venv/bin/activate
python src/main.py
```
